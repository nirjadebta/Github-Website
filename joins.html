Questions

Q1. Write a SQL statement to prepare a list with salesman name, customer name and their cities for the salesmen and customer who belongs to the same city.

Sample table: salesman

Sample table: customer

SELECT S.name AS Salesman,C.cust_name AS Customer,S.city FROM salesman S INNER JOIN customer C ON S.city = C.city.

Q2. Write a SQL statement to make a list with order no, purchase amount, customer name and their cities for those orders which order amount between 500 and 2000.

Sample table: orders

Sample table: customers

SELECT O.ord_no,O.purch_amt,C.cust_name,C.city FROM orders O INNER JOIN customer C ON O.customer_id=C.customer_id WHERE O.purch_amt BETWEEN 500 AND 2000.

Q3. Write a SQL statement to know which salesman are working for which customer.

Sample table: salesman

Sample table: customer

SELECT S.name AS Salesman ,C.cust_name AS Customer FROM salesman S INNER JOIN customer C ON S.salesman_id=C.salesman_id.

Q4. Write a SQL statement to find the list of customers who appointed a salesman for their jobs who gets a commission from the company is more than 12%.

Sample table: customer

Sample table: salesman

SELECT C.cust_name,C.city,S.name,S.commission FROM customer C INNER JOIN salesman S ON C.salesman_id =S.salesman_id WHERE S.commission>0.12;

Q5. Write a SQL statement to find the list of customers who appointed a salesman for their jobs who does not live in the same city where their customer lives, and gets a commission is above 12%

Sample table: salesman

Sample table: customer

SELECT C.cust_name, S.name,S.city AS "Salesman City" ,C.city AS "Customer City",S.commission FROM customer C INNER JOIN salesman S ON S.salesman_id = C.salesman_id WHERE (S.city <> C.city AND S.commission >0.12)

Q6. Write a SQL statement to find the details of a order i.e. order number, order date, amount of order, which customer gives the order and which salesman works for that customer and how much commission he gets for an order.

Sample table: salesman

Sample table: customer

Sample table: orders

SELECT O.ord_no,O.ord_date,O.purch_amt,C.cust_name,S.name AS Salesman, S.commission FROM orders O INNER JOIN customer C ON C.customer_id = O.customer_id INNER JOIN salesman S ON S.salesman_id = C.salesman_id;

Q7. Write a SQL statement to make a join on the tables salesman, customer and orders in such a form that the same column of each table will appear once and only the relational rows will come.

Sample table: customer

SELECT * FROM orders NATURAL JOIN customer NATURAL JOIN salesman

Q8. Write a SQL statement to make a list in ascending order for the customer who works either through a salesman or by own.

Sample table: customer

Sample table: salesman

SELECT C.cust_name FROM Customer C LEFT JOIN salesman S ON C.customer_id = S.customer_id ORDER BY C.cust_name

Q9. Write a SQL statement to make a list in ascending order for the customer who holds a grade less than 300 and works either through a salesman or by own.

Sample table: salesman

SELECT C.cust_name FROM Customer C LEFT JOIN salesman S ON C.salesman_id = S.salesman_id WHERE C.grade < 300 ORDER BY C.cust_name

Sample table: orders
